Add or Edit Redirect

How to add redirect

  1. Add which domain you wish to redirect in "Access URL" field.Eg.
    mjasion.pl for redirect only mjasion.pl
    *.mjasion.pl - catching all: www.mjasion.pl, blog.mjasion.pl...
  2. Add where you want to redirect domain. Begin address with http:// or https://
  3. Save redirect and mail me(marcin.jasion at gmail(dot)com) which domain you want to redirect :-)
  4. Add in your DNS record for this domain: CNAME nettool.herokuapp.com
